# 📚 PMBots: Custom Chatbot with Your Own Knowledge

Upload PDFs, DOCX, CSVs, Excel files, or website URLs — and chat with your own knowledge base.

Powered by:
- 🧠 OpenAI GPT

No installs. Just upload and ask!
